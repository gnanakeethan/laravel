http://framework.zend.com/license

Installable via the Artisan CLI:

	php artisan bundle:install zendframework

Auto-load the bundle in bundles.php:
 
	return array(
		'zend' => array( 'auto' => true),
	);

Useful links:
- homepage:      http://framework.zend.com
